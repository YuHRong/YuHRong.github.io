REAPER 主程序及扩展下载

REAPER 及其个扩展下载

一文带你从0开始下载REAPER 主程序和 SWS扩展，语言包以及OSARA扩展。

REAPER 主程序下载

进入REAPER 官网

进入Download下载页面

选择对应的系统进行下载。

windows

确定你的电脑是64位的还是32位的，选择对应的下载即可。

Windows 64-bit 或 windows 32-bit

windows 11/arm 架构的可以下载

Windows 11/arm64ec 测试版

Linux

这里可以下载 Linux 的程序。

macOS

这里可以下载 macOS 的个版本，按照自己系统选择下载即可。

主程序下载完成，接下来下载 SWS 扩展 和 语言包

SWS 扩展下载

返回 Reaper主页面

打开 Resources 资源，

点击 SWS REAPER Extension

我们在这里面去下载 SWS 扩展

同上，选择对应系统的安装包进行下载即可，

SWS扩展下载完成。

下载语言包

返回reaper主页面

进入Language Packs

下载语言包，如果想要使用英文可以忽略。

在语言包页面上只能看到有限的几个语言包可供下载。

下载其他语言得进入

REAPER Stash

在这里面我们就能看到很多翻译后的语言了。

你也可以在语言包页面下载语言框架自行翻译。初学者不建议这样做，

我们在这里去找翻译好的使用即可。

我们需要安装的是简体中文，一般带有ZHCN或者Chinese Simplified的都是简体中文包，

找到后点击下载即可

带有 zh_TW或者Chinese (Traditional)很大可能是简体中文，

若有需求也可进行下载。

若想要使用其他语言，也可以在这里寻找。

这里的语言包按需下载即可。

好了，语言包下载完成。

OSARA 下载。

如果你已经有配置好的操作方案，这一步完全可以跳过，往下找设置即可。

如果你是新接触 REAPER的初学者，是使用REAPER的盲人朋友，

强烈建议你下载OSARA扩展

他有一套已经做好的操作方案，可以让我们第壹次接触 REAPER的新手很快能上手使用。

如果从0开始配置一套使用方案的话，

对于 Reaper的新用户来说是比较麻烦的，

我们不妨先使用这一套扩展。

带对他的操作熟悉之后在逐渐替换成习惯的操作方案。

我从Cubase过来就是这样的。

盲人朋友使用他可以几乎无障碍的使用 REAPER进行混音 编曲 视频剪辑等操作

打开OSARA 开发快照

或者复制链接到浏览器中打开

https://osara.reaperaccessibility.com/snapshots/

他在这里只有 windows和mac两个频台的安装包，按需下载即可。

结尾

以上是REAPER 主程序和SWS 扩展和语言包以及OSARA扩展的下载，

我们将在下一张中介绍 REAPER及其个扩展怎样安装。
