Reaper 主程序及扩展安装

REAPER 主程序及扩展安装

在这一张中，我将使用 WINDOWS 系统的安装方式，带大家把上文中下载的 REAPER 的主程序和其他扩展给安装了。

Reaper 主程序的安装

首先打开你的下载目录，

若找不到在哪里，可以打开浏览器的下载管理，点击 在文件夹中显示或者 在文件夹中打开具体看浏览器的显示。

Reaper主程序的文件名大概长这样子，

reaper （版本号） _x64 或 （32）-install

如果设置了显示文件扩展名后面还有.exe扩展名。

Win11 arm 架构的文件名类似这样的。

reaper （版本号） _win11_arm64ec_beta（测试）-install .exe（扩展名

当然，若要显示扩展名，得在选项中显示文件扩展名。

这里我们可以右键点击以管理员身份运行

打开之后跳出来的是许可协议，若有时间，有精力你可以把他看完。

点击 I Agree按钮进入下一步。

这里出现的是选择安装路径，可以默认安装，也可以选择自定义目录。

若选择了其他路径务必要记清楚，后面有用。

后面有个 Portable install 复选框，安装便携版复选框。

若需要安装便携版勾选上他，安装目录务必选择其他路径，

若你有安装版在默认路径下，当你安装便携版的时候会发生什么我也不知道，我没尝试过。

大概什么也不会发生吧。

点击 next 下一步按钮

这里是选择安装组件，默认即可。

点击install安装按钮开始安装。

安装过程中你可以点击 Show details 查看安装详情。

安装完成之后点击 Close 关闭按钮完成主程序的安装。

这时他会提示你 Reaper 已经安装完毕，是否要运行，先点否。

因为你要是第壹次安装 REAPER 的话，第一次启动可能会有点慢，安装完所有扩展之后在启动他即可。

当 REAPER 主程序安装完成之后，SWS OSARA 和 语言包 的安装顺序就可以随意了，但安装完 语言包候他会启动 REAPER，建议把 语言包 放在最后安装。

SWS 的安装

其实跟 REAPER 的主程序安装几乎一样的，

出现 许可协议 就点 I Agree 我同意按钮。

路径选择不是便携版的就默认，继续点击Next下一步按钮。

最后点击Install 安装按钮。

完成安装。

OSARA 的安装。

OSARA 的安装几乎同上，

许可协议就点击 I Agree 我同意按钮。

唯一不一样的是点击 I Agree 我同意之后接下来的安装见面是问你是要安装便携版还是标准安装。

若安装了 便携版 的主程序，这里也要选择便携版，然后把目录选择到便携版 的路径下。

选择路径之后就 Install 安装。

选择标准安装之后点击下一步即可开始安装。

安装完成之后提示
Do you want to replace the existing key map with the OSARA key map?
New users are advised to answer Yes, which will completely replace your key map with a clean copy of the OSARA key map including all latest assignments.
Answering No will install OSARA w

简单翻译就是： 你是否需要使用 OSARA 的快捷键方案，点击是就将 Reaper 的默认快捷键方案修改为 OSARA 的快捷键方案。

若你是第壹次接触 REAPER 不妨点击是,这样可以免除你在刚开始打开REAPER之后一脸懵。

如果你有自己的操作方案就否吧。

语言包的安装

点击你在上文中下载的语言包

点击确定

在点一次确定

这时候就安装完成了，他开始启动REAPER，

若你的REAPER是第壹次安装的话，很有可能会启动一段时间，不必紧张，去泡杯茶，泡杯咖啡，上个厕所，洗个手，伸伸懒腰。他在扫描他自己的插件。等待一段时间即可。

好了，上面我们已经把REAPER 主程序 和 SWS 和 OSARA 以及语言包都安装完成。

下一张我们将开始进入REAPER的音频设备设置，

开始我们的REAPER之旅。
